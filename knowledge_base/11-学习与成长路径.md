# 11-学习与成长路径

> 核心原则: 市场是最好的老师，K线里有丰富的信息。看书只是一条微不足道的路径，观察市场才是正途。

---

## 一、学习投资的正确路径

### 原则1：市场是最好的老师

- 博主明确反对"开书单找秘笈"的学习心态。交易是一项技能，读书只是一条微不足道的路径。有人从未读过书也能做得很好，有人读了很多书却做不好一笔交易。好的书需要在合适的时候遇见，但市场永远是第一位的。
- 核心学习动作：看懂每一个走势是如何演变的，理解底部是怎样走出来的，顶部又是怎样走出来的。每次问自己：如果再来一次，你会选择在什么时候参与？
- **来源**: 087_学习的路径

### 原则2：反复观摩有代表性的牛股

- 每一支有代表意义的牛股都值得反复观摩。博主强调"做得好的要回顾，错过的更要回顾"。把K线拉回当时的情景，思考在当时什么是高、什么是低，什么是安全的买点。
- 从K线的五个数据（开、收、高、低、量）组合起来，读懂市场参与者的真实态度——比任何分析师的报告都诚实，因为它是真金白银堆出来的。
- **来源**: 087_学习的路径；P03_追高买入安全性_新易盛回顾

### 原则3：理解是操作的核心

- "理解，是一切操作的核心"——这是贯穿全部文章的最重要的元原则。理解市场的时候，尽量从价值和走势出发，而非从消息驱动出发。价值永恒，走势永远重复，这些是可以尝试去抓住的东西。而消息每一次都不一样，你很难准确判断。
- 市场很多时候是无法预测的，但确实有相当一部分是可以预判到的。每天多找一些个股和行业分析，想不明白也不用纠结，坚持分析几个月，一定会有所长进。
- **来源**: 087_学习的路径；088_交易之从不入门到精通

### 原则4：从K线到逻辑的双重理解

- 除了对K线的理解，还需要对逻辑的理解。虽然纯逻辑的思考意义不大，但对于很多上涨还是要尝试给出解释。比如为什么光伏最强的一直是逆变器？想明白后就能理解为什么微逆会成为后续热点。为什么消费、医药能在任何市场长期受喜爱？
- 用逻辑理解行业特性，用K线确认进出时机。两者相辅相成。
- **来源**: 087_学习的路径

## 二、博主的六年学习进化史（2020-2026）

### 阶段一（2020年）：框架搭建期 —— 从观察到记录

- 2020年是博主开始系统记录和分享的起点。核心文章《A股没有新鲜事》确立了"历史会重复"的基础信念，《找到那个不一样的点》建立了K线分析的核心理念。这一阶段他主要在看、在记录、在建立认知框架。
- 当年的A股主线是消费牛和新能源的起步。博客文章以观察市场规律为主，操作层面的内容较少，属于"先行理解市场"的阶段。
- 关键认知：所有的大行情都是从一次不起眼的反弹开始的。A股没有新鲜事，因为人性不变。
- **来源**: 001_A股没有新鲜事；011_找到那个不一样的点；087_学习的路径

### 阶段二（2021年）：体系成型期 —— 从理解到操作

- 2021年博主开始大量讨论买点体系、交易纪律和赚钱模式。《极简交易手册》系列、《牛市心法》、《怎样选出最强的股》等核心文章都出现在这一年。
- 这一年正式提出了"巴菲特+欧奈尔"的融合框架：巴菲特解决"买什么"，欧奈尔解决"什么时候买"。两个维度缺一不可。
- 关键突破：从"怎么看"到"怎么做"。顺势、突破、超跌三种核心赚钱模式成型。
- 这一阶段也遭遇了2021年春节后的核心资产崩盘，市场教育了他关于估值和趋势的关系。
- **来源**: 033_极简交易手册；015_怎样选出最强的股；052_我的两位老师

### 阶段三（2022年）：熊市考验期 —— 底部判断能力锤炼

- 2022年A股经历深度调整（沪深300全年跌超21%）。博主在这个阶段的文章大量讨论底部判断、防守策略、估值指标。《单底从来不容易》《见底还是崩盘》《一个中线必然挣钱的指标》等都在这一年。
- 在市场最悲观时（2022年10月底部），博主持续提示机会而非恐慌。反复强调格雷厄姆指数作为中线必然挣钱的指标。
- 关键突破：熊市中同样能赚钱。博主说"前两年指数那么难，其实我一直是把熊市当牛市来做，找的个股都是有铺垫、有蓄力、出现强势特征的票票"。
- 深度理解了"空头衰竭"和"力量逐渐转变"这两种最容易把握的转折信号。
- **来源**: 056_单底从来不容易；066_见底还是崩盘；079_一个中线必然挣钱的指标今天达到极限

### 阶段四（2023-2024年）：AI浪潮实战期 —— 知行合一的检验

- 2023年ChatGPT引爆AI行情，2024年牛市启动。这是博主理论框架在实战中大规模验证和迭代的阶段。
- 重点参与了CPO（新易盛、中际旭创等）、机器人（北特科技、埃夫特等）、低空经济（万丰奥威、宗申动力等）、存储（香农芯创、德明利等）、传媒等多个方向的完整周期。
- 《追高买入的安全性》《主升总是在相对高位展开》《多做简单的判断》《为什么是低空》等文章体现了对交易细节的深度思考。
- 关键突破："知道和拿住，其中仍然有不小的差距"。从理论到执行，经历了持仓管理的实战考验。
- 2024年初格雷厄姆指数再次进入历史级别低估区间（2.51），博主同步提示底部机会。
- **来源**: 085_主升总是在相对高位展开；062_多做简单的判断；P03_追高买入安全性_新易盛回顾；055_为什么是低空

### 阶段五（2025-2026年）：体系总结期 —— 从方法到心法

- 后期的文章以回顾和总结为主。《学习的路径》《交易之从不入门到精通》系统地梳理了整个投资方法论的底层逻辑。
- 博主开始更多讨论投资哲学、三条基本信仰（相信价值、相信重复、相信力量逐渐转变），从"教方法"转向"讲心法"。
- 《A股没有新鲜事》（2026年重写版）回顾了宏景科技翻倍案例，验证了"提前起步+指数调整期间稳住+保持横盘调整"的选股法则。
- 《追高买入的安全性+新易盛回顾》是收费文章合集的代表作，完整复盘了错过与抓住的全过程。
- **来源**: 087_学习的路径；088_交易之从不入门到精通；001_A股没有新鲜事（2026版）；P03_追高买入安全性_新易盛回顾

## 三、从亏钱到稳定盈利的具体路径

### 原则5：稳盈20%的门槛 —— 四条核心纪律

- 博主在《交易之从不入门到精通》中给亏钱的朋友写了一份"实操手册"：
  1. **只做简单题的容易部分**：找到相对强势、趋势向上的方向，再找到这个方向上比较强的个股，在趋势股的低吸位置买入。
  2. **取消关注大部分大V**：放弃关注大部分热门股。很多人关注的个股并不适合每个人操作。包括博主在雪球说到的标的，有很多也并不适合一般人操作。
  3. **不接近买点不要随便买**：虽然高手复盘以打板为主，但如果你还没实现稳定盈利，先戒掉这些习惯。或许会错过很多机会，但也根除了吃到大面的可能。
  4. **不贪婪，有压力就跑**：不接受洗盘。对于没有实现稳定盈利的人，资金量不算大，先试一下，一个月30个点到手，今年够本了，去等下一个机会。
- **来源**: 088_交易之从不入门到精通

### 原则6：均线低吸法入门 —— 最易掌握的盈利模式

- 股市盈利的方法千千万（龙头、超跌反弹等），但博主认为最容易掌握的是趋势股的低吸。所有走出了明显的、肉眼可见的上升趋势之后的个股，首次贴近10日线都是非常好的买点。
- 具体操作：找到上升趋势的股票，等待首次回踩10日线并走出小杯柄形态。即使判断失误，遵守好的资金管理原则，这种趋势向上的均线低吸几乎永远可以将亏损控制在非常有限的范围内。
- 弱点在于周转率低一点，但经常说：每次20%，4次翻倍；每次30%，3次翻倍；每次40%，2次翻倍。低吸的方式成功率不低，赔率也比较高。
- **来源**: 088_交易之从不入门到精通

### 原则7：成功的模式反复做

- 博主在《成功就是重复做》中讲了一段生动的经历：润泽科技某种程度上就是前两天掌阅科技的重复——头天长上影线，第二天中等幅度高开，就是一个上车的机会。"这就是一种成功的模式反复做，成功了两次，那么下次还敢。"
- 学好一两种模式，再配上适当的仓位管理和资金管理策略，基本上就不用羡慕别人了。
- **来源**: 061_成功就是重复做；085_主升总是在相对高位展开

### 原则8：统计自己的交易 —— 焦点管理

- 博主分享了一位朋友的经历：这位朋友统计了自己所有的交易记录，发现亏损最多的是开盘追高，而开盘后半小时内的半路板及一个小时内的低吸成功率较高。于是就专心做这两种交易，放弃所有其他类型。当交易聚焦之后，成功率甚至得到了更大提升。
- 操作指引：定期统计找到亏损最多和成功率最高的来源，砍掉低成功率聚焦高成功率。操作后验比关注"买什么"重要得多。
- **来源**: 054_一位朋友的暴赚体验

### 原则9：学习高手的真正收获

- 博主学习过不少交割单后认为：很多人学到后面就学到坑里去了，因为一个时期的市场氛围、个股特性各有不同，如果当时没有深度参与，事后再去复盘，很难重现当时的场景。
- 学习的最大收获不是高手选中了多少牛股，而在于"多赚少亏，净值保持向上"。很多人在极低位置多次卖飞过牛股，但一方面他们不会介意在后面再买回来，另一方面，就算不买回来，对净值的向上也不会有太大影响。
- **来源**: 088_交易之从不入门到精通

## 四、核心心态与纪律

### 原则10：错过不悔，卖飞不气

- 这是博主反复强调的交易心态。卖飞了不要紧，通过努力寻找机会，用短线的心态来对待交易，无畏卖飞。关键是整体净值保持向上。
- 博主多次以自己做错的案例示人，包括新易盛的错过：第一买点觉得短期反弹确定性不如热乎的机器人，第二买点舒泰神、一品红等医药股正在风口比它刺激。遗憾但不纠结。
- **来源**: 088_交易之从不入门到精通；P03_追高买入安全性_新易盛回顾

### 原则11：不要浪费任何一次振荡中枢

- 博主多次在文章中提到这一原则。振荡中枢（横盘整理区间）不仅是持仓观察的窗口，更是寻找新买点的时机。每一次脱离中枢都是重新上车的机会。
- 常山北明是典型案例："每一次脱离盘整的中枢都是一次重新上车的机会"。
- **来源**: 015_怎样选出最强的股；043_牛市心法

### 原则12：市场没有错，都是你的错

- 股价与业绩出现背离的时候，如果你不能理解，那么大概率市场知道得比你更多一些。要么有些事情你还不知道，要么你反应太慢，要么你缺乏耐心。
- 不要把亏损归因于市场、政策或主力。每一笔亏损交易都是"学费"，关键是学费不能白交——必须从中提炼出可以复用的经验。
- **来源**: 087_学习的路径

## 五、从"知道"到"做到"的跨越

### 原则13：知道和拿住的差距

- 博主反复强调"知道和拿住，其中仍然有不小的差距"。在存储方向完整经历了香农芯创的跌破10日线和次日拉回、果断买回并加仓的过程后，他感慨：回头看下跌时帖子里说的理由，是不是觉得知道和拿住确实有不小的差距。
- 知道是认知层面，拿住是执行层面。两者之间的鸿沟只能通过反复实践和复盘来弥合。
- **来源**: 015_怎样选出最强的股；P03_追高买入安全性_新易盛回顾

### 原则14：预判不重要，重要的是行情出现时如何上车

- "对于一个大行情，你的预判其实意义也不大，所以不要去做无谓的吵架。更重要的是如果行情出现如何去上车。"这是博主从交易者角度给出的核心建议。
- AI应用吵了无数次，大部分时候是过山车体验。但作为交易者，每天去和别人吵架、站队，那是和钱过不去。如果应用出现了稳健的走势，就找机会上车；如果没有，就看戏。
- **来源**: P02_从交易角度看待AI应用；043_牛市心法

### 原则15：多看是什么，少问为什么

- 投资人最常犯的错误是过度分析原因而忽略现象本身。盘面已经告诉你强弱（均线排列、量价关系、调整形态），不需要去猜测背后的原因。
- 正确的方式是：看到现象 → 判断强弱 → 制定应对方案 → 根据市场反馈调整。
- 需要知道的是"现在是什么状态"——趋势向上还是向下？强还是弱？正在加速还是减速？
- **来源**: 043_牛市心法；P02_从交易角度看待AI应用；087_学习的路径
