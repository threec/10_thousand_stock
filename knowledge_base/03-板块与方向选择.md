# 03-板块与方向选择

> 核心原则: 选对方向是选对个股的前提 —— 先在正确的河流里游泳，再挑最漂亮的浪花。

---

## 一、三步法：指数定向→板块定位→个股定买点

### 原则1.1：从上到下的三层过滤

- 博主在《怎样选出最强的股》中提出的"三步法"是整个选股体系的操作框架：第一步，判断指数环境（大盘是否安全、风格在什么方向）；第二步，找到当前资金最活跃的板块（板块排位、赚钱效应）；第三步，在强势板块中选择符合买点条件的个股。这三步不能颠倒，不能从个股倒推板块（"我买了这个股所以这个板块好"）。
- 每一步都有具体标准：
  - 第一步标准：指数站上20日均线（短期）、年线方向向上（长期）、成交量温和放大
  - 第二步标准：板块内有3只以上的个股同步走强（非孤立个股行情）、板块成交量明显放大、板块指数站稳重要均线
  - 第三步标准：个股在板块内强度排前3-5、符合四种买点模式中的至少一种（见05文档）
- 正例：2023年AI行情——指数环境回暖→板块层面光模块/算力集体走强→选新易盛等领涨股上车
- 反例：不关注板块环境，看到某个股票涨了就追进去，结果发现整个板块都在下跌，只有它是"庄股"在硬撑
- **来源**: 015_怎样选出最强的股

### 原则1.2：三步法的优先级不可颠倒

- 博主强调，很多人做反了——先看到一只股票觉得不错（第三步），然后去查这个股票在什么板块（第二步），最后发现大盘环境不对已经重仓了（第一步）。正确的顺序永远是：天时（指数）→地利（板块）→人和（个股）。如果天时地利都不对，个股再好也不是出击时机。
- **来源**: 015_怎样选出最强的股；043_牛市心法

## 二、牛市重势，熊市重质

### 原则2.1：不同环境下的不同选股逻辑

- "牛市重势，熊市重质"是博主引用最多的一句选股格言。牛市中的核心逻辑：趋势就是最大的基本面。只要一只股票处于明确的上升趋势中、板块效应显著、资金持续流入，即使PE偏高也可以参与——因为牛市情绪会给出更高的估值溢价。熊市中的核心逻辑：价值是唯一的保护伞。只有那些PE合理、不亏损、经营稳健的股票，才能在大盘下跌时提供相对安全，并且在反弹时率先启动。
- 操作指引：在指数全面多头时（牛市），选股偏向"强度"维度——涨停板多、趋势强、盘子适中；在指数空头或震荡时（熊市/平衡市），选股偏向"价值"维度——PE低、业绩稳定、有分红。
- 正例：2020年牛市阶段，消费股PE动辄50+但依然持续上涨，"重势"策略有效；2022年熊市阶段，低估值高股息策略显著跑赢大盘，"重质"策略有效。
- 反例：2022年在熊市中追高成长股（重势轻质），结果深度套牢；2020年在牛市中死守低估值银行股（重质轻势），错失大行情。
- **来源**: 015_怎样选出最强的股；P01_跌下来应该会有一些机会

### 原则2.2：势和质不是非此即彼

- 博主的最佳实践是寻找"既有势又有质"的方向——即基本面有支撑（不亏损、PE合理区间、行业发展逻辑清晰）且走势上正在走强（均线多头、放量突破、板块效应显著）的板块。这种"双优"方向往往能走出大级别的趋势行情，而纯概念炒作（有势无质）或深度价值陷阱（有质无势）都走不远。
- **来源**: 036_多选那些有生命力的方向

## 三、有生命力的方向

### 原则3.1：基本面与走势双验证

- 博主提出"有生命力的方向"这个概念，用于区分真主线和假热点。一个有生命力的方向必须满足两个条件：第一，基本面上有支撑逻辑（产业趋势、政策支持、业绩改善预期等），不是纯粹的概念炒作；第二，走势上有真金白银的流入（成交量放大、领涨股强势、板块内轮动健康），不是一日游行情。
- 双验证的检验标准：利好消息出来时，板块是否真金白银地涨？（真涨说明市场认可）利好消息消化后，板块是否缩量横盘而不是暴跌回落？（横盘说明资金锁仓）指数回调时，板块是否抗跌或快速收回？（抗跌说明资金护盘意愿强）
- **来源**: 036_多选那些有生命力的方向

### 原则3.2：领涨股不能全是亏损股和底部票

- 博主给出了一条硬性筛选标准：看一个板块的领涨股结构。如果领涨的全是亏损股（没有业绩支撑的炒作）和长期下跌后刚反弹的底部票（超跌反弹而非趋势反转），那么这个方向大概率没有生命力，不能作为中线配置。真正有生命力的方向，领涨股中必须有2-3只"基本面不错+走势强势"的中军标的。
- 具体判断方法：打开板块涨幅榜，看前5名的股票 —— 它们的PE是多少？它们过去一年的走势是上升趋势还是下降趋势？它们的净利润是否为正？如果5只里有3只以上是亏损的或底部反弹的，降低仓位或放弃。
- **来源**: 036_多选那些有生命力的方向

## 四、下跌检验上涨成色

### 原则4.1：真金还需火炼

- 博主在《真金还需火炼》中详尽阐述了一个核心鉴别方法：大市下跌是检验板块和个股成色的最好时机。在大盘下跌期间，你的持仓或关注的板块如果表现出色（不跌反涨、跌幅明显小于指数、跌下去后迅速收回），这就是真正强势的信号，说明里面的资金不想走、外面的资金还想进。
- 鉴别的三个级别：
  - 最优：大盘跌，板块涨 → 最强的强势信号，资金逆势涌入
  - 良好：大盘跌3%，板块跌1%以内 → 资金护盘意愿强，合理强势
  - 一般：大盘跌，板块同步跌 → 无独立行情，随波逐流
  - 风险：大盘跌1%，板块跌3%以上 → 弱势，里面的资金在趁机出逃
- 操作指引：大盘每次跌2%以上的那天晚上，检查你持仓股票的跌幅，如果跌幅小于大盘的2/3，可以拿住；如果跌幅大于大盘的1.5倍，考虑减仓。
- **来源**: 029_真金还需火炼

### 原则4.2：迅速收回才是真强势

- 博主特别强调"迅速收回"这四个字。在大盘下跌中抗跌的板块不少（可能是护盘资金），但真正有生命力的方向是——不仅在大盘跌的时候抗跌，更重要的是在大盘企稳后能迅速创新高。如果只是抗跌但大盘反弹时它不涨，说明里面是防御型资金而不是进攻型资金，这个方向的攻击力不够。
- 验证时间窗口：大盘止跌后3-5个交易日内，强势板块应该收回失地并尝试创新高。如果超过一周都收不回来，则强势程度存疑。
- **来源**: 029_真金还需火炼

## 五、放弃方向的两个条件

### 原则5.1：赚钱效应明显不如其他方向

- 博主在《坚守科技与传媒》中给出了放弃一个方向的明确条件：第一，市场上出现了另一个赚钱效应明显更强的方向（涨幅更大、持续性更好、参与体验更佳）。在这种情况下，资金会从旧方向流出涌入新方向，继续坚守旧方向的性价比下降。
- 赚钱效应的量化判断：新方向的涨停板数量 > 旧方向的3倍以上；新方向的板块涨幅连续3天 > 旧方向；新方向出现了批量创新高的股票而旧方向没有。
- **来源**: 082_坚守科技与传媒

### 原则5.2：该方向彻底赚不到钱

- 第二个放弃条件更简单：你自己在这个方向上反复操作却反复亏钱。这不是一种"耻辱"，而是市场在告诉你这个方向和你现阶段的能力圈不匹配。博主的态度是实用主义的：能赚到钱的方向才是好方向，不管你多么看好它的基本面。如果你看好的方向让你持续亏损，先退出来，等行情好做了再回来。
- 量化标准：同一个方向连续3次操作都亏钱（不论金额大小）→ 暂停该方向；或者持有该方向的主要股票超过2周没有任何盈利 → 重新评估。
- **来源**: 082_坚守科技与传媒

## 六、同时关注最强和最弱

### 原则6.1：最强是方向，最弱是风险

- 博主在《2022年10月报》中提出一个简洁但实用的观察框架：每次复盘时，找出全市场最强的3-5个板块和最弱的3-5个板块。最强的告诉你钱在往哪里流（进攻方向），最弱的告诉你风险在哪里聚集（需要回避的方向）。两者同等重要。
- 为什么最弱也很重要：因为如果你持有的股票恰好属于最弱板块，即使个股基本面再好也很难逆市上涨。而且最弱板块的持续下跌可能引发连锁反应（如质押风险、基金赎回压力等），最终拖累大盘。
- **来源**: 089_2022年10月报

### 原则6.2：最强与最弱的背离信号

- 当最强的板块开始走弱（龙头股不再创新高、高位放量滞涨），而最弱的板块开始走强（底部放量反弹、不再创新低），这是风格可能切换的先兆。不需要等到最弱变成最强才反应，出现背离就可以开始调整仓位。
- **来源**: 089_2022年10月报；078_两次转折一个重要反思

## 七、板块轮动节奏

### 原则7.1：三类板块的属性划分

- 博主将A股的主要板块分为三类，每类有各自的运行节奏和操作特点：
  1. **进攻型**（半导体、AI、机器人、新能源、券商）：弹性大、波动大、行情来了涨得猛、行情结束跌得也猛。适合在指数环境好的时候重仓参与。操作策略：追涨不追高（起涨时果断进入，涨多了不追），见顶信号出现后坚决撤退。
  2. **防御型**（医药、消费、公用事业）：弹性小、波动小、趋势稳定、适合长期配置。在指数环境不确定或偏弱时提供避风港。操作策略：低吸不追涨（回调到支撑位买入），长期持有做波段。
  3. **周期型**（有色、煤炭、化工、券商）：强周期性，行情来得快去得快，时机最重要。操作策略：关注大宗商品价格和宏观经济拐点，左侧轻仓试探 + 右侧确认加仓。
- 轮动规律：一轮完整的行情通常是 防御型先稳 → 进攻型启动 → 周期型补涨 → 全线冲顶。但这只是大致节奏，每一轮都有差异，不可机械套用。
- **来源**: 030_牛市回忆录；P02_从交易角度看待AI应用

### 原则7.2：主线和轮动的辩证关系

- "主线"是指能够持续引领行情的核心板块，通常有强有力的基本面逻辑支撑（产业趋势、政策支持等），能吸引最多的资金参与，持续时间长达数月至一年以上。"轮动"是主线行情中出现的正常现象——主线休息的时候，资金会短期溢出到其他板块，但随后又会回流主线。
- 操作策略：主仓位（60%以上）应该配置在主线上，因为主线有最强的确定性和持续性；辅助仓位（20-30%）可以参与轮动，但要及时止盈，因为轮动的持续性通常不如主线。不要在轮动中追高买入，不要在主线休息时卖出主线去追轮动。
- 识别主线的方法：第一，板块连续2周以上每日成交额位居全市场前3；第二，板块内有超过5只股票在2周内创新高；第三，指数回调时该板块跌幅显著小于大盘且有快速收回。三个条件同时满足，大概率是主线。
- **来源**: 030_牛市回忆录；043_牛市心法；082_坚守科技与传媒

### 原则7.3：主题投资的参与时机

- 对于主题投资（如低空经济、合成生物、商业航天等），博主的态度是"可以参与，但要清楚这是主题而非主线"。主题行情的特点是爆发力强但持续性差，往往在政策或事件催化下快速拉升，然后迅速回落。
- 参与条件：第一，出现标志性催化事件（政策发布、重大技术突破等）；第二，板块内出现3只以上连续涨停的股票（说明游资认可）；第三，大盘环境不是单边下跌（至少是震荡或上升）。三条都满足时可以小仓位参与，但止损要设得更紧（-8%即走），因为主题退潮的速度远快于主线。
- **来源**: 055_为什么是低空

## 八、估值修复模式

### 原则8.1：估值修复的四要素

- 博主用"低空经济"作为案例，总结了估值修复行情（区别于产业趋势行情）的四个要素：①经营稳健（过去3-5年业绩稳定，没有大起大落）、②长期被市场忽视（过去1-2年股价横盘或下跌，无人关注）、③估值处于历史低位（PE/PB都在底部区间）、④出现一个新题材触发市场重新关注（政策利好、行业事件等）。
- 估值修复和产业趋势的关键区别：估值修复的空间相对有限（通常修复到历史均值附近就差不多了），持续时间也较短（1-3个月）；而产业趋势行情一旦确立，空间和持续性都更大。操作策略上，估值修复见好就收，产业趋势可以拿久一点。
- 正例：2024年低空经济板块 —— 许多标的长期被忽视+估值低+政策催化来袭 → 一波修复行情
- **来源**: 055_为什么是低空

## 九、方向选择的实操流程

### 原则9.1：每日方向扫描法

- 博主在多篇文章中暗示的每日复盘流程（关于方向的扫描）：
  1. 看涨跌幅榜：涨幅前20的板块（关注共性）、跌幅前20的板块（关注风险）
  2. 看成交额榜：成交额前10的板块（资金主战场在哪）
  3. 看涨停板分布：涨停股在哪些板块集中出现（游资态度）
  4. 看创新高/新低：哪些板块有批量创新高的股票
  5. 看持续性：连续3天以上走强的板块（过滤一日游）
- 完成上述扫描后，心中应该有一个清晰的答案：当前最值得关注的方向是A、B、C，需要回避的方向是X、Y、Z。
- **来源**: 综合015、029、089等多篇文章

### 原则9.2：方向确认后的跟踪清单

- 一旦确认了核心关注方向，博主建议建立以下跟踪清单：
  - 板块指数的关键均线位置（5日、10日、20日、60日、年线）
  - 板块内龙头股的走势状态（哪个在加速、哪个在休息、哪个在补涨）
  - 板块每日成交量的变化趋势（放量还是缩量）
  - 板块相对于大盘的强弱变化（相对强度指标）
  - 板块内部分化程度（龙头和中后排股票的差距是否在扩大）
- 这些数据可以帮助你在方向内部选择最优的个股和买点时机。
- **来源**: 综合015、036、039等多篇文章

## 十、方向选择中的常见陷阱

### 原则10.1：后视镜选板块 —— 涨了才看到

- 博主提醒投资者最常见的错误：当一个板块大涨了30-50%之后才决定"这个方向太好了，我要布局"。这时候你看到的所谓"投资逻辑"其实是市场涨了之后帮你编造的故事。真正的方向选择应该在板块刚开始走强（刚脱离底部、刚开始放量）的阶段就敏锐地识别出来，而不是等到了加速阶段再冲进去。
- 对抗方法：每天复盘时关注那些"刚刚开始放量走强但还没有大涨"的板块。具体来说，关注板块指数刚刚站上20日均线且20日均线刚刚开始走平的板块，而不是那些已经连续大涨两周以上、全网都在吹捧的板块。
- **来源**: 015_怎样选出最强的股；036_多选那些有生命力的方向

### 原则10.2：眼花缭乱 —— 每个方向都想做

- 市场每天都有涨停的股票，但绝大多数与你无关。博主强调"放弃"是一种核心能力。如果你试图参与每一个看起来不错的方向，你的注意力会被稀释到零，最终每个方向都是浅尝辄止，赚不到大钱。正确做法是：确认1-2个主方向，深度跟踪，放弃其他所有看起来"还行"的方向。
- 强度测试：如果一个方向你无法在5分钟内说清楚它的核心逻辑、龙头股是谁、板块处于什么阶段——那么这个方向就不属于你，你应该放弃它。不是因为你能力不行，而是因为你的注意力已经分配不过来了。
- **来源**: 082_坚守科技与传媒；038_10倍比两倍更简单
