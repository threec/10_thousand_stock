# 13-经典案例研究

> 核心原则: 理论的价值在于应用。每一支有代表意义的牛股都值得反复观摩，每一次成功或错过的操作都值得深入复盘。

## 案例1：新易盛完整操作复盘 —— 最详细的单一案例

### 背景

2024-2025年AI硬件牛市的核心品种。新易盛作为CPO光模块龙头，从底部到高点涨幅数倍。博主在收费文章《追高买入的安全性+新易盛回顾》中对这笔操作做了完整复盘，坦诚地分析了看到却没有做到的全过程。

**来源**: P03_追高买入安全性_新易盛回顾

### 第一买点：120周线底部的反弹（2024年4月）

**识别过程**：
- 2024年4月下旬，博主发帖指出新易盛可能已见底。
- 核心判断依据（技术面）：
  - 周线回踩了120周线（长期生命线）
  - 同时触及长期趋势线和前期顶部形成的多重支撑
  - 结合英伟达走势判断：不管是新易盛还是达子，如果再下去，AI纪元以来的贡献就会全部回吐，"不涨很不合适"
- 核心判断依据（逻辑面）：AI硬件的基本面变化是真实的——CPO公司确实实现了"脱胎换骨"的业绩增长，与AI应用停留在三年前的状况完全不同。

**后续走势**: 见底后新易盛没有立即走好，而是经历了较长时间的底部盘整。

**为什么错过**: 博主当时觉得短期反弹的确定性不如"还热乎的机器人"。他把资金配置在了当时更活跃的方向上，没有在底部建立仓位。

**教训**: 
- 120周线的底部是一个极具价值的信号，即使短期不走好，也应该建立观察仓
- "确定性和弹性"之间的权衡：底部抄底的确定性低，但一旦成功后成果会很丰厚
- 博主事后的反思："这两年做顺势的股票做得太多，做起来也轻松。当一些长期关注的股票跌到黄金坑时，如果有能力抄到底，也未尝不可"

### 第二买点：突破新高+回踩年线（2024年10月）

**识别过程**：
- 博主标注的关键位置：新易盛刚刚实际突破历史新高，同时回踩了年线和半年线，均线全部掉头向上，开始趋势上行。这是一个"异常点"之后的确认——之前的上涨被验证不是随机的。
- 这是典型的"中期走强+短期调整结束"的结构。

**为什么错过**: 当时舒泰神、一品红等医药股正在风口，比新易盛"刺激"。博主选择了短期弹性更大的方向。

**教训**:
- "永远应该保持空杯心态，给主升阶段的个股更高的地位"
- 在多个机会同时出现时，需要对照自己的体系来排优先级，而不是被短期刺激左右
- 新高突破+均线回踩是一个非常经典的买点信号，属于"简单题"的范畴

### 第三买点：业绩确认加速（2025年1月）

**识别过程**：
- 业绩被市场接受，进入加速上涨阶段
- 从底部起来约4倍，其中"容易被把握住的应该有大约一倍多的空间"

**操作**: 博主在这个阶段参与了短线，但没有描述重仓持有的情况。后续新易盛继续上涨，成为雪球热股。

**总结反思**:
- 博主在结尾的感悟最为深刻："这一波AI，说遗憾也不算太遗憾，毕竟最近市场热闹，机会并不少。但是给了我一些新的感悟。"
- 核心教训：在长期关注的股票跌到黄金坑时，如果有能力抄到底，也未尝不可。逆势抄底虽然成功率低一些，但成功后成果会很丰厚。

**方法论关联**:
- 买点体系（三步买点法）：`05-买入时机与买点体系.md`
- 趋势判断（杯柄突破）：`04-个股筛选体系.md`
- 阶段划分（长期底部→趋势启动→加速）：`02-市场环境评估体系.md`
- 心理执行（知道vs拿住）：`11-学习与成长路径.md`

## 案例2：CPO板块完整周期 —— 多个月的主线操作

### 背景

2023年ChatGPT引爆AI行情后，CPO（共封装光学）成为A股AI硬件中最核心的分支。从2023年初到2024年，整个CPO板块经历了多轮上涨和调整，博主在整个周期中进行了反复参与和深度跟踪。

**来源**: 039_为什么今天机器人涨不过算力；054_一位朋友的暴赚体验；P02_从交易角度看待AI应用；060_反弹行情都怕高

### 阶段一：初期识别（2023年初）

**识别逻辑**:
- AI的降临确实让CPO公司实现了"脱胎换骨"的业绩增长——这是CPO区别于AI应用最根本的地方
- 走势特征：提前起步，在指数调整期间能稳住或保持横盘调整
- "为什么cpo能够越涨越高，因为AI的降临，确实让其中一些公司，实现了脱胎换骨。而应用，大部分公司还停留在三年前的样子。"

**参与**: 在早期阶段开始跟踪，包括中际旭创、新易盛等核心品种。

**来源**: P02_从交易角度看待AI应用

### 阶段二：主升期（2023年4-6月）

**走势特征**:
- 板块出现明显的"主升浪"特征：历史新高或至少一年新高，中短期无套牢盘
- 核心股票始终保持在10日均线之上
- 与英伟达走势高度相关："去年CPO的每一段主要涨幅，都是发生在英伟达主升期间"

**操作手法**: 均线低吸法——上升趋势中首次贴近10日线是非常好的买点。核心品种中际旭创的趋势非常标准，调整缩量、上涨放量。

**来源**: 039_为什么今天机器人涨不过算力

### 阶段三：调整期应对（2023年下半年至2024年初）

**走势特征**:
- 2023年6月21日，中证传媒、科大讯飞、CPO同步见顶
- Cpo指数跌到接近半年线，博主提示"中际旭创110附近应该要有反弹"

**策略调整**:
- 博主在较长时间内没有碰CPO，转向了传媒和其他科技股
- 给出了明确的择时："上一波过于用力，这一波如果调整不够充分的话可能不如别的之前没被爆炒过的方向"
- 2024年初CPO大阳后横盘两个月，博主回补仓位："虽然有点贵了，但是想想一直持有CPO的话两个月没赚钱了，时间成本也是要钱的"

**来源**: 054_一位朋友的暴赚体验；082_为什么要继续坚守科技与传媒

### 阶段四：2024年春季反弹与分化

**走势特征**:
- 中际旭创从底部翻倍，回到前期高点附近，处于"纠结"状态：是牛市延续还是双顶？
- 新易盛、天孚通信相对更强
- 2024年3月27日CPO大幅跳水，博主判断为资金调仓而非基本面变化

**博主的应对**:
- 指出中际旭创与上证指数相关度极高，两者的振荡是同步的
- 对比历史上的重要底部（998、1849、2440），判断上涨时间和反弹周期，认为此轮上涨时间尚短（仅30个交易日）
- 结论："我认为持有是没有问题的。这个逻辑很通顺。"

**关于调整的深度思考**:
- "不管你是多有经验的交易者，市场上永远会有让你意想不到的情况，适当的分散、仓位的控制永远是保命的手段。"
- 即使看对了方向，也要为意料之外的波动留有余地

**来源**: 060_反弹行情都怕高

### 阶段五：2025-2026年的独立走势

**走势特征**:
- "最近CPO的走势我认为是一个脱离外围、独立加速的走势"
- 与英伟达不再完全相关，走出了中国科技股的独立行情
- 摆脱对标思维成为A股走出自己节奏的关键一步

**来源**: 041_摆脱对标思维是件好事；062_多做简单的判断

### 操作精髓总结

1. **初期**：识别行业基本面的"脱胎换骨"vs"原地踏步"——这是判断持续性的根本
2. **主升期**：均线低吸，大胆持有，不恐高
3. **调整期**：注重性价比——充分回调后再进入，或在核心支撑位低吸
4. **分化期**：去弱留强，核心品种优于杂毛
5. **退出**：当高位巨阴+横盘整理后不能继续新高时警惕

**方法论关联**:
- 板块筛选（"有生命力的方向"）：`03-板块与方向选择.md`
- 买点节奏（均线低吸+突破追入）：`05-买入时机与买点体系.md`
- 仓位管理（分散+分批）：`06-仓位与资金管理.md`
- 卖出信号（加速+分化）：`07-卖出与止盈止损.md`

## 案例3：医药板块底部判断 —— "中线必然挣钱"的案例

### 背景

中证医药从2021年高点持续下跌至2024年初，跌幅深、时间长、估值创历史纪录。博主在多个关键时间点对医药板块做了底部判断和跟踪。

**来源**: 023_你会加仓什么样的医药；048_医药还能买吗；079_一个中线必然挣钱的指标今天达到极限

### 关键节点一：2022年5月的第一次底部判断

**判断依据**:
- 博主首次说"医药可以买"是在2022年5月
- 当时精准抓住了短期低点
- 但如果一直持有不动到2024年中，浮亏-30%

**教训**: 短期底不等于长期底。医药的调整周期远超最初预期。这说明了为什么要等趋势确认、为什么要分批买入。

**来源**: 048_医药还能买吗

### 关键节点二：2022年10月的月线级别底部

**判断依据**:
- 月线级别的大通道支撑
- A股全面调整后的系统性底部
- 博主在医药指数月线通道下轨处做了第二次底部判断

**后续**: 这一判断的支撑在2023年6月被彻底破坏，医药进入了"没有什么支撑的区域"，进入了最黑暗的下跌阶段。

**来源**: 048_医药还能买吗

### 关键节点三：2023年9月的空头衰竭信号

**判断依据** (这是博主认为最经典的一次底部判断):
- 场外稳健资金买入医药基金
- 医药的跌幅、时间、估值都创下了历史纪录
- 日线在低位出现连续下跌甚至超跌信号
- 辅以趋势线分析
- 估值角度：格雷厄姆指数的极限信号
- 走势角度：空头衰竭——下跌缩量、K线实体变小、下影线增多、对利空反应钝化

**博主的判断逻辑**:
"这是一次空头衰竭的买入。这一波医药的跌幅、时间以及估值都创下了历史纪录，日线在低位上出现了连续下跌甚至出现超跌信号，再辅以趋势线分析，那就是一个值博率很高的买点。这就是基于价值和走势的分析。"

**反驳纯逻辑思维**:
"基于逻辑分析，当时就看到很多人说有集采的利空，不能买。你再去看看集采是从哪一年开始的，而医药的高点出现在哪一年，再看集采之后医药公司整体盈利水平怎么样。就知道，纯逻辑的思考无法判断高点，也无法判断低点。可以说实战中基本没啥用。"

**来源**: 087_学习的路径

### 关键节点四：2024年1月格雷厄姆指数极限

**判断依据**:
- 格雷厄姆指数（股票盈利收益率/十年期国债收益率）达到2.51，A股历史上仅2008年底部出现过类似数值
- 历史上任何一次当格雷厄姆指数>2.2时进入市场，几乎都可以在一年内取得较不错的收益
- 博主称其为"一个中线必然挣钱的指标"

**医药指数状态**:
- 医药指数已长期跌破10年低点
- 博主建议在这个位置"我也不想卖"，"割肉可能是不太合适的"
- 对定投者：暂停定投或拉大定投区间
- 对场内交易者："目前的力度只能说是加入关注，涨势不够炸裂，领涨的票也不够漂亮。等几个周不跌回来再开始重视也来得及。"

**来源**: 079_一个中线必然挣钱的指标今天达到极限；048_医药还能买吗

### 关键节点五：2025年医药全面看好

**判断依据**:
- 创新药分支已经提前走得非常强劲
- 中证医药的均线走势显示整体处于上涨的起涨阶段
- 博主明确表示"医药，接近于全面看好"，"都还没开始上涨"
- 舒泰神新低反弹、常山药业新高，说明医药在这附近是阶段低点

**具体策略**:
- 分散买一些有特色、有趋势、有一定业绩支撑的个股
- "在标的并不很明确的时候，适当的追高买入是非常常见的操作手法"
- 每一次买入或加仓，尽量要买入符合预期或高于预期的，而不是买低于预期的

**来源**: 023_你会加仓什么样的医药；P03_追高买入安全性_新易盛回顾

### 操作精髓总结

1. **不要精确择时底部**：博主在医药上的多次底部判断中，有精准的也有超前的。关键是等到信号出现再去介入。
2. **多重信号交叉验证**：估值（格雷厄姆指数）+ 走势（空头衰竭）+ 趋势线 + 板块内个股信号
3. **定投和短线完全不同策略**：定投需要耐心和在低位持续投入，短线需要等趋势确认
4. **纯粹的逻辑分析无法判断顶底**：这是最核心的教训——"纯逻辑的思考无法判断高点，也无法判断低点"

**方法论关联**:
- 估值体系的运用：`02-市场环境评估体系.md`
- 底部判断法则：`01-投资哲学与底层信仰.md`（空头衰竭+力量逐渐转变）
- 行业分析框架：`15-行业与赛道分析框架.md`（医药赛道模板）

## 案例4：传媒板块坚守过程 —— 持仓心理的实战

### 背景

从2023年AI行情启动开始，传媒（中证传媒指数）就与AI走势高度相关——中证传媒、科大讯飞、CPO在2023年6月21日同步见顶。此后传媒经历了长达数月的下跌，许多持有者经历了巨大的心理考验。

**来源**: 081_传媒还能做吗；082_为什么要继续坚守科技与传媒；061_成功就是重复做

### 阶段一：2023年传媒的辉煌与转折

**辉煌时期**:
- 2023年上半年，传媒是AI行情中弹性最大的方向之一
- 游戏、出版、影视全面开花，赚钱效应极好
- 博主在传媒上有大量仓位和成功的交易

**转折点**:
- 2023年6月21日，传媒指数见顶，与科大讯飞和CPO同步
- 随后开启长达数月的下跌调整

**来源**: 082_为什么要继续坚守科技与传媒

### 阶段二：深度回撤中的心理考验（2023年下半年）

**市场状态**:
- 传媒持续下跌，跌破博主之前所说的目标位
- 大量粉丝反复询问"要不要放弃传媒"
- 外部出现了许多"传媒不行了"的声音

**博主的回应与坚持**:

**问题1: 要不要放弃传媒与泛科技？**
回答："当然不会。"理由是评价一个方向强不强不光要看自身表现，也要看历史的进程。传媒虽然跌破了目标位，但是整个A股都跌破了。新能源、医药的走势还不如传媒。

**问题2: 什么时候放弃这个目标？**
给出了两个明确的退出条件：
1. 当市场上出现别的方向持续赚钱效应明显比这一方向更强的时候
2. 当科技股赚不到钱的时候

**问题3: 为什么继续看好传媒？**
- 传媒仍是近期赚钱效应最好的方向之一
- 仔细观察最近走强的科技传媒股，"一旦开始反转，强度都是比较大的"
- 数据要素分支已经提前反转走强，走势到位了
- 出版类股息率可观（中文传媒7.8%、山东传媒/凤凰传媒超6%），有底线安全

**来源**: 082_为什么要继续坚守科技与传媒

### 阶段三：传媒的防守与进攻（2024年初）

**防守端——出版业**:
- 出版类公司账面资金充裕、现金流漂亮、盈利稳定、暴雷风险小、估值不高
- 高股息率提供了下档保护
- 博主称出版为"传媒中的防守派"，即使下跌底部也不会太深

**进攻端——AI概念的弹性品种**:
- 游戏、影视、数据要素等方向，弹性大但波动也大
- 操作上"涨到有压力的位置先减一点再说，偏谨慎一点"

**来源**: 081_传媒还能做吗

### 阶段四：百花纷呈的爆发（2024年3月）

**爆发契机**:
- AI概念持续发酵，kimi等新应用引爆市场
- 传媒板块百花纷呈，大涨
- 博主此前布局的掌阅科技、中原传媒等收获丰厚

**博主的冷静评估**:
- 掌阅科技："我觉得就是蹭热点多一些，我估计最多还有一两个板就差不多了"
- 中原传媒："做了一个漂亮的平台，有业绩断层，又有大单和消息催化。厉害了。"
- 中文传媒等传统出版股："股灾受伤很轻，而现在也走出了慢牛的走势"

**来源**: 061_成功就是重复做

### 操作精髓总结

1. **不要因为短期的下跌就放弃一个方向**：要先看整个市场环境，你持有的方向相对其他方向是什么状态
2. **设定明确的退出条件**：不是凭感觉说要放弃，而是要看是否有别的方向更强、是否方向本身赚不到钱
3. **了解持仓的"性质"**：同一个板块内，有防守型（出版-高股息）和进攻型（游戏-高弹性），要用不同的仓位和节奏来对待
4. **在坚守中保持灵活**：不是死扛不动，而是"整体看好、灵活轮动"——涨到压力的减仓，跌到支撑的加仓

## 案例5：宏景科技与"A股没有新鲜事"

### 背景

2026年4月，博主在重新修订的《A股没有新鲜事》中回顾了宏景科技的翻倍行情，作为"A股没有新鲜事"这一核心信条的最新验证。

**来源**: 001_A股没有新鲜事

### 识别过程

**发现逻辑**:
- 博主在一个月前（2026年3月）转发了一篇文章，提及宏景科技
- 判断方法"完全是一样的道理，提前起步，然后在指数调整期间能稳住，或者保持横盘调整，这就是很好的"
- 同样的方法还适用于存储、CPO、锂电等方向

**核心筛选条件**:
- 在指数调整期间，个股能保持横盘/小跌/甚至逆势走强
- 提前于板块和指数起步
- 横盘调整的形态紧凑、成交量有序

**来源**: 001_A股没有新鲜事；032_一些可能会耽误赚钱的偏好

### 操作过程

**时间线**:
- 博主发文后第二天：竞价买入宏景科技
- 一个月后：翻倍

**为什么能抓住**:
- 博主说这不是运气，而是把"存储时的分析方法"完整复制了一遍
- 选股原则的一致性：提前起步 + 指数调整期间保持横盘 + 算力租赁的预期差更大（"很少有人看好"）

### 关键教训

1. **方法的一致性和可复制性**：博主明确说"很多方法都是固定的，而方向也有持续性，没有必要每天复一次盘"
2. **预期差的价值**：算力租赁相比CPO、锂电，"很少有人看好"——这恰恰是预期差最大的时候
3. **不需要每天看盘**：博主强调"很多方法都是固定的"，好的机会出现时有足够的时间来识别和介入
4. **华尔街没有新鲜事，A股也没有**：100年前利弗莫尔悟出的道理，在今天的A股依然成立。因为"市场反复重演这件事发源于每一个参与者的贪婪与恐惧，没有任何力量能轻易改变"

## 案例6：万泰生物压力位分析 —— 精确的技术分析实例

### 背景

2024年初，万泰生物在九价HPV疫苗预期的推动下快速上涨。博主的压力位分析展示了精确的技术分析如何为操作提供指导。

**来源**: 079_一个中线必然挣钱的指标今天达到极限

### 操作过程

**识别压力位**:
- 万泰生物涨到120周线附近，与之前智飞生物的见顶位置完全一致
- 博主精确对比："之前智飞生物的周线是涨到120周线附近见顶。万泰今天同样是涨到120周线附近。"

**精准的减仓建议**:
- 博主给出明确建议："我觉得万泰这个位置还是应该要减仓应对。"
- 理由不是"感觉太高了"，而是有精确的技术依据：120周线在智飞生物身上已经被验证为关键压力位

**对后续走势的两种预判**:
- 如果短期快速拉回去："应该要越涨越减"——不要被快速拉升迷惑
- 如果在这个位置做充分的中期调整："说不定有机会继续向上"——给调整后的行情留下期待

### 关键教训

1. **压力位不是玄学**：博主用智飞生物的实际案例作为对照，给压力位的判断提供了实证依据
2. **对标历史走势的有效性**：同一板块中，前一只股票的顶部特征往往能用于预判后一只股票
3. **为什么冲高回落不用找太多理由**："不用想太多理由，我觉得最主要的原因就是昨天大阴线，进去做反包的人早上看到冲不上去，整体行情又不好，就反手砸了"

- 技术分析：`04-个股筛选体系.md`（均线与关键位置）
- 卖出规则：`07-卖出与止盈止损.md`（压力位减仓法则）
- 对标思维的危险与局限：`14-常见错误与行为偏误.md`（对标思维的陷阱）
