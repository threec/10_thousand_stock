# 15-行业与赛道分析框架

> 核心原则: 选对赛道是超额收益的起点。识别什么样的方向"有生命力"、什么样的行业正在发生什么级别的变化，比选个股更重要。

---

## 一、"有生命力的方向"评估体系

### 原则1：三大筛选维度

博主在《多选那些有生命力的方向》中，系统总结了判断一个方向是否"有生命力"的三个维度：

**维度一：基本面 —— 主业有点东西 + 插上想象的翅膀**

- 公司不能是纯亏损的纯概念股
- "主业有点东西"：有扎实的业务基础，经营稳健，不会暴雷
- "插上想象的翅膀"：有一个题材或新方向给估值插上翅膀——AI、机器人、低空等
- 博主的经典表述："它们之中有不少都是传统制造业。可以说以前的业务不够性感，但是业绩要说很坑那倒也不至于。"
- 反面案例：很多纯炒作的题材，领涨股全是亏损股，上去给你不舒服的机会买，下来卖不出

**维度二：走势 —— 长期走势不错 + 核心股走出长牛慢牛**

- 板块里有一部分核心股票走出了长牛慢牛的走势，是关键信号
- 以机器人板块为例：北特科技"里面是有长期资金的"，虽然不一定要买，但是"在板块里有一部分较核心的股票走出了长牛慢牛的走势，可以给你参与这个板块提供一些信心"
- 走势稳定性比弹跳更关键："能稳住，远比冲得快重要"

**维度三：题材 —— 领涨股不能全是亏损股+底部票**

- 博主的底线：领涨股不能全是亏损股。如果领涨的都是一些基本面极烂的票，这个方向的持续性值得怀疑
- 强度维度：领涨股不能全部是底部刚反弹的票（底部票强度不够）
- 正例：低空经济——万丰奥威大涨几倍后PE也才30-40倍，估值回归是可行的
- 反例：车联网（v2x）——"领涨股从价值来看，几乎全是亏损股。从强度来看也都是底部票，强度是不够的。"

**来源**: 036_多选那些有生命力的方向；052_我的两位老师

## 二、科技赛道分析模板

### 2.1 AI硬件（CPO/光模块/服务器/存储）

**技术路线确定性**:
- AI硬件是确定性最高的赛道之一：AI的降临让CPO公司实现了"脱胎换骨"的业绩增长
- 数据增长是确定性的，"看到了持续的指数级的增长"
- 与AI应用的根本区别：硬件有真实的业绩支撑，应用大部分公司还停留在三年前的样子

**核心筛选逻辑**:
- 提前起步 + 指数调整期间保持横盘 = 最好的品种
- 结构紧凑度："涨的时候流畅，跌的时候跌得少的个股就属于紧凑的结构"
- 博主以并行科技（日线）和神州数码（月线）为例展示了"节奏感好"的走势

**估值天花板**:
- 硬件股不讲市盈率讲产业趋势：博主在新易盛涨了数倍后仍然参与短线
- 但博主也会提醒风险：当CPO成为雪球热股、吸引全场眼球时，加速显然很难持续
- 跟踪的重要催化剂：英伟达业绩、AI资本开支、新技术路线

**操作节奏**:
- CPO的每一段主要涨幅发生在英伟达主升期间（2024年以前）
- 2025年开始CPO出现脱离外围、独立加速的走势
- "加速显然很难持续，成本低的随意，成本高的只能短线考虑"

**来源**: P02_从交易角度看待AI应用；039_为什么今天机器人涨不过算力；062_多做简单的判断；P03_追高买入安全性_新易盛回顾

### 2.2 半导体/存储/PCB

**景气度判断**:
- "半导体我说得很清楚了，根本没法挑，这是一个方向的整体景气。不要天天问哪一个最好，而是更多地关注当你想下单的时候，哪个的位置更合适。"
- 半导体月线"一直在大级别的上升过程中，每次到120月线都会出现非常强力的支撑"

**走势特征**:
- 精测电子："每一次大涨后，都会有一个横盘等待的过程"
- 长川科技："走强了很久，每一次到关键位置的时候，总是能拉起"
- 兆易创新："非常漂亮的杯柄走势"。博主判断后面可能会走上流畅的上涨（5日线不破）

**存储特定的预期差**:
- 博主2025年明确重点看好存储方向
- 香农芯创是大级别突破，德明利和江波龙同理
- 存储的板块效应不如CPO强，需要做"去弱留强"
- 扩散路径：核心存储票先涨 → PCB/CIS等后续跟进

**风险信号**:
- "这一目标已经不是太远"——中证500在通道上轨
- ETF大量卖出是热度信号
- "最好不要疯牛，疯牛的结果只会很惨"

**来源**: P02_从交易角度看待AI应用；015_怎样选出最强的股

### 2.3 机器人/人形机器人

**参与的底层逻辑**:
- "主业有点东西，然后插上了想象的翅膀" —— 最像2024年低空经济的板块
- "从机器狗的成熟，到具身机器人，再到上市量产，这是一个漫长的过程"
- "值得挖掘的东西还很多" —— 产业链深、细分多、轮动空间大

**走势特征**:
- 2024年底：在指数下跌中保持稳定，北特科技、中坚科技等核心票走势紧凑
- 调整幅度：主流个股调整幅度在20%以内（比软件方向更扛跌）
- "50-60日线是从牛市起点以来一直比较有效的支撑"
- 博主反复强调的一个错误："应用之类的底部涨起来我并不着急...如果真有大行情，让它先翻一倍也行"

**操作节奏**:
- 初期可以分散布局，等核心票浮出水面后再集中
- "在板块里有一部分较核心的股票走出了长牛慢牛的走势"是持续参与的信心来源
- 调整期减仓的信号："在指数上涨，而它们明显跟不上了之后"

**主升浪判断标准**（通用法则）:
1. 应该是历史新高，或一年新高，至少是大半年的新高——中短期无任何套牢盘
2. 在相当长一段时间（至少一二十天）内保持在主要均线之上（5日最好，10日也行）
3. 均线需要是向上发散的
4. 板块效应取决于有多少个股符合以上条件

**来源**: 039_为什么今天机器人涨不过算力；029_真金还需火炼；032_一些可能会耽误赚钱的偏好；036_多选那些有生命力的方向

## 三、医药赛道分析模板

### 3.1 创新药/CXO/中药/医美

**政策周期的影响**:
- 集采是医药板块最大的政策变量。但博主指出一个关键事实：集采从2018年就开始了，而医药的高点出现在2021年；集采后医药公司整体盈利水平并没有显著下降
- "纯逻辑的思考无法判断高点，也无法判断低点"
- 医药指数的下跌时间和幅度均创历史纪录，但集采并不是暴跌的唯一甚至主要原因

**研发管线的价值**:
- 创新药的价值很难用传统估值方法来确定——"创新药的价值显然很难确定。即使对于能确定价值的股票，当它处于调整周期的时候，短期内也并没有半点安全性可言"
- 但不能因此否定价值的存在：万泰生物的九价疫苗预期是推动股价的核心逻辑
- 百济神州从底部起来的完整过程表明：趋势走出多头排列后回踩10日线才是安全买点

**估值修复的触发条件**:
- 格雷厄姆指数达到极端值（>2.2，2024年1月达到2.51为历史次高）
- 场外稳健资金开始入场（2023年9月医药基金买入信号）
- 板块内领涨品种开始出现底部形态的改善
- "等几个周不跌回来再开始重视也来得及"

**博主的核心判断**:
- 2024年初："医药，接近于全面看好"。中证医药的走势显示"都还没开始上涨"
- 创新药分支提前走得非常强劲
- 操作上不挑重点——"有特色，有趋势，有一定业绩支撑的个股，没有那么好挑...只能分散买一些"

**来源**: 079_一个中线必然挣钱的指标今天达到极限；087_学习的路径；048_医药还能买吗；023_你会加仓什么样的医药

## 四、周期类分析模板

### 4.1 有色/煤炭/石油/券商

**供需周期的位置判断**:
- 博主对煤炭、石油等旧能源品种的参与有明确的买入逻辑：上升趋势首次碰10日线+对上证50的底部判断=关键均线买入具有安全性
- "如果这些东西都补跌，只会让大家对市场更绝望"——强势品种的走势是市场信心的风向标
- 旧能源品种即使在熊市中也有结构性机会

**价格信号的领先性**:
- 中国石油的上涨——博主判断"可能没有结束"
- "走势好好的都崩了，走势差的买了又有啥意思"——强势品种在系统性风险面前是更好的选择

**券商=牛市放大器**:
- 博主对券商的态度非常谨慎："券商我是暂时不打算看"
- 但认可券商作为牛市放大器的作用：券商拉升同时导致低空和AI跳水，说明券商吸走了市场的流动性
- 参与券商需要等确认的牛市到来，而非提前埋伏

**来源**: 066_见底还是崩盘；070_中国石油的上涨可能没有结束；054_一位朋友的暴赚体验

## 五、出海作为隐藏主线

### 5.1 出海赛道的识别与操作

**为什么出海是隐藏主线**:
- 博主在2024年5月就提出了出海线大概率没有走完
- "出海这条线不管是逻辑还是走势都是说得过去的"
- 出海的特点是"不够显眼，但持续性强"——不是市场最热闹的方向，但一批又一批出海公司走出了慢牛

**选股逻辑**:
- 具备海外经营能力，有国际竞争力的中国制造业公司
- 经营稳健，估值合理，有明确的业绩增长逻辑
- 走势上：调整至关键均线（半年线、年线）后重拾上升趋势

**代表性品种及走势**:
- 宇通客车：博主最关注的出海标的之一，在半年线位置有支撑，"趋势尚未形成，但仍然可以关注"
- 嘉益股份：在半年线位置"似乎是重回上升趋势了"
- 豪迈科技："最近走势很强劲"
- 匠心家居："调整至年线后重拾上升走势"
- 三星医疗："同样的意思"

**与其他主线的区别**:
- 出海不是爆发力最强的方向，但是持续性最好的方向之一
- 不是每天都有催化剂，但趋势一旦形成，可以持有较长时间
- "光讲指数见底意义不大，还要考虑个股的情况"

**来源**: 046_出海是一条隐藏的持续主线

## 六、估值修复模式识别

### 6.1 "不够好但还不错"的估值修复

**模式定义**: 博主在《为什么是低空》中系统阐述了这一模式：
公司经营稳健，不会暴雷，不会出大问题，没有明显亮点，但是由于多年来被忽视，趁一个题材，有人进来做一波。这批"虽然不够好但还不错的票票"在进行估值提升。

**模式特征**:
1. 经营稳健：主业正常运营，不暴雷，基本面没有重大问题
2. 长期被忽视：多年来没有表现，估值处于历史低位
3. 估值低：PE在十几倍甚至个位数，具有估值修复空间
4. 趁一个题材来一波：有一个催化剂或题材触发重新估值（不一定需要题材有多靠谱）

**经典案例**:
- **鸣志电器**：非常普通的公司，在一个较窄的行业有一定地位，上市后多年没表现。直到机器人题材出来，"一口气涨了七八倍"
- **工业富联**：很普通的制造业大厂，没有什么亮点，但有一定地位，趁AI题材来一波
- **万丰奥威**：好好做生意的公司，深耕行业多年，上涨前PE只有十几倍，六年沉寂后几乎回到顶部
- **宗申动力**：在摩托车行业干了三十多年，自2015年大牛市后再无表现，上涨前PE个位数，"算是一次猛烈的估值修复"
- **苏交科**：国资、破净股、上涨前股息率3%左右、PE十几，"涨上几十个点修复一下估值，也算是合理"

**博主的核心洞见**:
- "这样的描述，其实是中国不少传统制造业的现状"
- "中国领先于世界的就是制造业，但是利润率低，地位不高，面临技术升级压力"
- "现在的A股，确实是有点让人觉得难办啊，整体看上去并不贵，但增长快，够便宜，有一定想象空间，短期没被爆炒过筹码很干净的标的，真是不多"
- "以后，可能要少想一下'这个题材靠不靠谱'，而更多地去思考，板块内的标的，是否存在可以上涨的基础"

**来源**: 055_为什么是低空

## 七、行业判断的实战框架

### 7.1 行业比较与选择逻辑

**什么是"好的行业"**（博主的四个观察维度）：

1. **持续性**：方向有没有持续性，不是一日游行情。博主对日常层出不穷的新题材"九成都是忽略的"。"评价一个行情值不值得做，至少要看一次涨落的表现"
2. **板块容量**：板块要大，有足够的标的可以参与。"分支较小，别的芯片整体离走好还有一定距离"
3. **赚钱效应**：方向里能不能赚到钱、有没有顺畅的买点。如果"不给机会买的票非常少，只看敢不敢去"
4. **轮动潜力**：方向内部有没有扩散的空间：从核心票到边缘票、从硬件到软件、从老龙头到新面孔

**行业排序的实战运用**:
- 博主对其关注方向的排序"并不表示短期涨幅，而更多的是对未来持续性的一个猜想"
- 排序是动态调整的，需要根据走势的反馈不断修正
- 当多个方向同时有机会时，比较的不是谁的弹性更大，而是谁的持续性更好

**来源**: 029_真金还需火炼；039_为什么今天机器人涨不过算力；043_牛市心法

### 7.2 行业底部判断框架

**必须同时满足的条件**（从医药板块底部判断中提炼）：

1. **估值极端**：格雷厄姆指数>2.2，或板块PE处于历史极端低估区间
2. **跌幅足够**：长期大幅下跌，创历史纪录的跌幅和时间
3. **走势衰竭**：下跌缩量、K线实体变小、下影线增多、对利空反应钝化
4. **领涨品种出现**：板块内开始有股票走出趋势扭转的形态
5. **时间窗口**：至少要等"几个周不跌回来"再做确认

**关键提醒**: 
- "单底从来不容易"——第一次反弹后往往还会有回踩
- "短线抓得准确但中长线下去30%是完全可能的"
- 底部是一个过程，不是一个点

**来源**: 079_一个中线必然挣钱的指标今天达到极限；056_单底从来不容易；048_医药还能买吗

## 八、科技与传媒的特殊关系

### 8.1 AI行情中的科技-传媒联动

**相关性实证**:
- 中证传媒与AI走势高度相关：传媒指数与科大讯飞、CPO在2023年6月21日同步见顶
- "去年把chatGPT板块的走势和传媒股走势叠起来，相关度是极高的"
- "这一轮传媒就是AI周期"——传媒是AI在A股的重要映射

**传媒内部的细分定位**:
- 出版（防守型）：高股息、稳定盈利、低估值，是传媒中的"避险资产"
- 游戏（弹性型）：与AI的业绩关联最直接，弹性最大
- 影视/数据要素（事件驱动型）：需要消息催化，爆发力强但持续性弱

**来源**: 082_为什么要继续坚守科技与传媒；061_成功就是重复做；081_传媒还能做吗
